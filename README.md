# Tâi-gí Telex 輸入法 demo

**[繁體中文](README.md) | [English](README.en.md) | [Tâi-gí](README.nan.md)**

Tâi-gí Telex 台羅拼音輸入法線上體驗網站。

**線上體驗：** https://telex.kahiok.com

## 功能說明

此輸入法使用類似越南 Telex 輸入法的聲調機制來輸入台羅拼音。聲調符號可以放在母音後面或每個音節的結尾。

### 聲調按鍵對照表

| 按鍵 | 聲調  | 台羅範例 | 台羅結果 | 白話字範例 | 白話字結果 |
| ---- | ----- | -------- | -------- | ---------- | ---------- |
| v    | 第2聲 | tev      | té       | hoov       | hó͘         |
| y    | 第3聲 | khooy    | khòo     | pay        | pà         |
| d    | 第5聲 | langd    | lâng     | kaud       | kâu        |
| w    | 第7聲 | phinnw   | phīnn    | ciunnw     | chhiūⁿ     |
| x    | 第8聲 | titx     | ti̍t      | lokx       | lo̍k        |
| q    | 第9聲 | tsangq   | tsa̋ng    | ianq       | ia̋n        |

### 其他功能按鍵

因為這幾個字母放著不用實在太浪費了，所以我擅自把他們拿來作其他用途：

| 按鍵 | 功能（台羅 / 白話字） | 台羅範例 | 台羅結果 | 白話字範例 | 白話字結果 |
| ---- | --------------------- | -------- | -------- | ---------- | ---------- |
| z    | 子音：ts / ch         | zo       | tso      | zit        | chit       |
| c    | 子音：tsh / chh       | ci       | tshi     | coew       | chhōe      |
| f    | 連字符                | taif     | tai-     | taif       | tai-       |
| nn   | 鼻音（僅白話字）      | -        | -        | sann       | saⁿ        |
| oo   | 半鼻音（僅白話字）    | -        | -        | poo        | po͘         |

### 使用範例

- 輸入 `tai[d]` → 顯示 `tâi`
- 輸入 `tâi[f]` → 顯示 `tâi-`
- 輸入 `tev` → 顯示 `té`
- 輸入 `khooy` → 顯示 `khòo`
- 輸入 `langd` → 顯示 `lâng`
