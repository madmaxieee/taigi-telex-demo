Sú-iōng Tâi-gí Telex su-ji̍p-huat lâi phah Tâi-lô phing-im ê sòo-tuānn.

🔗 **Suànn-tíng thé-giām：** https://telex.kahiok.com

## Suat-bîng

Tsit ê su-ji̍p-huat sú-iōng Ua̍t-lâm Telex su-ji̍p-huat ê hong-sik lâi tsú Tâi-lô phing-im ê siann-tiāu. Siann-tiāu hû-hō ē-sái tuì tī guân-im āu-piah a̍h-sī kui-ê im-tsat ê kiat-búe.

### Siann-tiāu khí tùi-tsiáu-pió

| Khí | Siann-tiāu | Lē              | Kiat-kó |
| --- | ---------- | --------------- | ------- |
| v   | Tē-2 siann | tev             | té      |
| y   | Tē-3 siann | khooy           | khòo    |
| d   | Tē-5 siann | langd / ladng   | lâng    |
| w   | Tē-7 siann | phiwnn / phinnw | phīnn   |
| x   | Tē-8 siann | tixt / titx     | ti̍t     |
| q   | Tē-9 siann | tsaqng / tsangq | tsa̋ng   |

### Kî-thann kong-lîng khí

In-uī tsit kuí-ê jī-bó bô iōng sit-tsāi siūnn thó-tsè, sóo-í guá kā in the̍h lâi tsò kî-thann ê kong-lîng:

| Khí | Kong-lîng   | Lē   | Kiat-kó |
| --- | ----------- | ---- | ------- |
| z   | Tsú-im: ts  | zo   | tso     |
| c   | Tsú-im: tsh | ci   | tshi    |
| f   | Liân-jī-hû  | taif | tai-    |

### Sú-iōng lē

- Phah `tai[d]` → hián-sī `tâi`
- Phah `tâi[f]` → hián-sī `tâi-`
- Phah `tev` → hián-sī `té`
- Phah `khooy` → hián-sī `khòo`
- Phah `langd` a̍h-sī `ladng` → hián-sī `lâng`
